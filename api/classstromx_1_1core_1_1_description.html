Detailed Description

Description of an ID to data variant map.

Instances of this class associate an ID with a data variant. Input nodes, output nodes and parameters of operators are characterized by such a description. In case of input and output nodes the data variant defines which data type can be passed to an input node or is to be expected from an output node. In case of parameter it characterizes the data type of the respective paramter. In addition, a Description can have a name which is purely informal and does not have any influence on its behavior.
