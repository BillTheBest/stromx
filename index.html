stromx

Tools for near real time processing of data with a focus on industrial imaging.

Introduction

The stromx project currently consists of two subprojects: stromx and stromx-studio. More information can be found on the introductory page of the stromx wiki.

Binaries

Prebuild packages of a current development version of stromx and stromx-studio for Linux and Windows exist. The Linux packages are hosted at openSUSE Build Service and launchpad. Note that while the packages are updated regularly they are not always based on the most recent sources. Moreover, the packages for different distributions are not synchronized. The current Ubuntu packages might e.g. have been built from newer sources than the latest Windows installer.

Sources

Documentation

The current stromx documentation contains several tutorials and a description of the library API.

Build instructions

The common wiki of both projects provides useful information about the necessary prequisites for building stromx and stromx-studio on Linux and Windows. It also contains build instruction for both platforms.
