Unstable

Prebuilt packages of a current development version of stromx and stromx-studio for Linux exist. The Linux packages are hosted at openSUSE Build Service and launchpad. Note that while the packages are updated regularly they are not always based on the most recent sources. Moreover, the packages for different distributions are not synchronized. The current Ubuntu packages might e.g. have been built from newer sources than the Fedora packages.
