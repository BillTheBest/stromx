About

Stromx-web is an open source solution to create and deploy vision applications. It runs on virtually every Linux based device from low-cost embedded platforms and smart cameras to full blown industrial PCs connected to several high performance cameras. Vision applications — so called "streams" — are assembled by connecting operators in a visual editor.

The settings of each operator can be adjusted on-the-fly while the stream is executed. Because all interaction is done in an HTML5 based web interface the vision device can reside in remote locations or be integrated in an headless client.

Nevertheless the actual image processing takes place in a robust C++ runtime and can take advantage of parallel architectures. Stromx operators can be easily created using the stromx SDK. Operators for many functions of the OpenCV library have already been implemented and are part of stromx-web. Also an operator library for the Raspberry Pi camera and its GPIOs exists.
